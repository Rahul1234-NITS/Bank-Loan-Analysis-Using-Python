# Bank Loan Analysis

An end-to-end exploratory data analysis (EDA) project on a bank loan dataset, covering key lending KPIs, monthly trends, and breakdowns by state, term, employment length, loan purpose, and home ownership.

## 📊 Overview

This project analyzes **38,576 loan applications** from a US-based lending dataset (2021) to uncover insights around loan funding, repayment, and risk (good vs. bad loans).

## 🔑 Key Metrics

- **Total Loan Applications:** 38,576
- **Total Funded Amount:** $435.76M
- **Total Amount Received:** $473.07M
- **Average Interest Rate:** 12.05%
- **Average DTI (Debt-to-Income):** 13.33%
- **Good Loans (Fully Paid / Current):** 86.18% of applications
- **Bad Loans (Charged Off):** 13.82% of applications

## 🛠️ Tools & Libraries

- **Python** — pandas, numpy
- **Visualization** — matplotlib, seaborn, plotly (treemaps)
- **Environment** — Jupyter Notebook

## 📈 Analysis Covered

- Overall loan portfolio KPIs (total & Month-to-Date)
- Good loan vs. bad loan breakdown
- Monthly trends — funded amount, amount received, applications
- State-wise loan distribution
- Loan term-wise breakdown (36 vs. 60 months)
- Employment length-wise breakdown
- Loan purpose-wise breakdown
- Home ownership-wise breakdown (treemap)

## 📁 Files

- `Bank_loan_analysis.ipynb` — Main analysis notebook
- `financial_loan.xlsx` — Source dataset
